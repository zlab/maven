## 用法
``` xml
<repository>
  <id>zlab</id>
  <name>zlab</name>
  <url>https://raw.githubusercontent.com/zlab/maven/master/</url>
</repository>
```

## 安装
``` bash
cd ~/.m2/repository
git init
git remote add origin git@github.com:zlab/maven.git
git fetch
git checkout master
```
